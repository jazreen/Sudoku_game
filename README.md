# Sudoku_game
Sudoku game - Work in progress!

*Personal Summer Project, started in August 2023*

Over the summer, I took an initiative to self-learn Python (enrolled in Harvard CS50's Introduction to Programming in Python) and this project emerged as a means to reinforce my learning. 
Right now, I'm working on creating a sudoku game solver where given an unsolved 2D array of the game, the program will return the solved array. I'm still working on it, meaning it's not completely functional but I'm hopeful to get there soon.
It currently has one sample matrix that's being used to debug the code, but in the future I wish to change it so that the user will be able to upload their own matrix through file I/O.

I hope to continue working on the project over the school term, however considering the typically demanding nature of schoolwork, I anticipate that progress will take place at a more gradual pace.
My ultimate goal is to develop a GUI and transform the project from a solver to an interactive sudoku game.
